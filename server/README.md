notes api
==============

This is basic api for posting and showind notes made using NodeJS Ð°nd MongoDB.

Routes are: 

// Get all notes
[GET] : [server-url]/api/notes

// Get specific note
[GET] : [server-url]/api/note/:id

// Add new note
[POST] : [server-url]/api/notes/new

// Delete note
[GET] : [server-url]/api/notes/delete/:id
